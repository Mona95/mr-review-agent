# 🤖 MR Review Agent

> AI-powered code review bot for GitHub Pull Requests.
> Automatically posts structured feedback on every PR.

## What it does

When a pull request is opened — this bot automatically:
- Fetches the PR diff
- Analyzes it with AI (Groq + Llama 3.3)
- Posts a structured review comment with bugs,
  security issues, style suggestions, and fixes
